NgVerse provides three simple methods to add elements to your application:

Using NgVerse Schematics

The easiest way to add a element to your project is by using NgVerse schematics. This approach automates the process for you. Simply specify the element you want, and schematics will handle the setup.

Downloading the Component Files

Alternatively, you can download the element files directly:

Navigate to the desired element's page on the NgVerse documentation site.

  • Click the Download ZIP button.
  • Extract the downloaded ZIP folder.
  • Import the extracted folder into your project.
  • Once added, the element will be ready to use.

Copy/Paste the Source Code

Every element page includes a source code tree that you can manually copy and paste into your project. Follow these steps:

  • Navigate to the desired element's page.
  • View the source code tree.
  • Copy the necessary files or code snippets.
  • Paste them into your project in the appropriate locations.
